# Streamer Compliance Prompt

Run this as the final gate before presenting generated live-commerce copy.
Also read `references/script_evaluation_rubric.md`; rubric P0 failures are not just style issues and must be repaired before delivery.

## Block Or Rewrite

- Extreme advertising claims: absolute best, first, national-level, 100%, permanent, guaranteed cure, zero risk.
- Medical or health-effect claims without permitted evidence.
- Financial return, investment, income, or risk-free claims.
- Fake scarcity: false stock, false countdown, false sell-out pressure.
- Fake price promises: invented original price, lowest price, platform-wide comparison, unauthorized subsidy.
- Fabricated proof: certifications, tests, factory source, celebrity use, buyer results, GMV, rankings.
- Emotional coercion and audience stratification: "不买后悔一辈子", "中国人不骗中国人", "懂的人直接上，不懂别问".
- Malicious comparison or attack: "市面上 99% 都是垃圾", insulting competitor brands, or implying only this product is reliable without proof.

## Mark As Truth-Dependent

- Inventory and countdown.
- Coupon, discount, activity price, gift.
- Material composition and origin.
- Shipping, return, after-sales policy.
- Sales volume, ranking, repurchase, conversion, feedback.

## Output

If safe, append a short compliance note. If unsafe, rewrite the risky phrase and explain what fact is needed.

## Rubric Hard-Gate Repairs

- If the script only lists selling points, add a real scene, action/process, comparison or perceivable detail.
- If the script lacks product-audience fit, explicitly state who should buy and why.
- If the script lacks information richness, add available product display, parameters, fit recommendation, usage instructions, authority proof, and after-sales categories without fabricating.
- If the script lacks category atmosphere, add category anchors such as beauty skin feel / finish / application process, apparel fabric / fit / scene, food texture / flavor / eating scene.
- If the script lacks conversion, add a clear action path and at least two truthful conversion reasons.
